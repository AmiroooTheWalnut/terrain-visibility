# Qt project

Qt version 6.8LTS Creator version 14.0.2

Required Qt libraries: QML, Quick, Quick3D, Graphs, GUI
Required external libraries:
	OpenCV
		Installation:
			Linux command: sudo apt install libopencv-dev
